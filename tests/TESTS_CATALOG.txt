================================================================================
  COMPILER TEST CATALOG  (OOP project — calculator.exe test sources)
================================================================================
Run from this folder after building main\Compiler\calculator.exe :

  PowerShell:  .\run_all_tests.ps1
  Manual:      ..\main\Compiler\calculator.exe test_XX_name.txt

Each test uses print(expr); so you can see results. The program also prints
"--- Results ---" and the symbol table dump at the end.
